## Available Scripts

In the project directory, you can run:

### `yarn serve`

Runs the server.

### `yarn start`

Runs the app.

### `yarn test`

Launches the test runner.

### `yarn testWatch`

Launches the test runner in interactive watch mode.

### `yarn lint`

Runs a lint including type check.

### `yarn build`

Builds the app for production to the `build` folder.

### `yarn eject`

Expose CRA configuration (one way operation)
